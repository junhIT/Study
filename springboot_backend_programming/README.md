## STS Lombok 미적용
**1. lombok 설치파일 실행**

STS 좌측 project explorer -> Project and External Dependencies(jar파일 폴더)
-> lombok.jar파일 우클릭 -> Run as -> Java Application -> lombok 실행

**2. lombok 설치**

Specify location... 클릭 -> STS 실행파일 선택 -> 사용하는 프로그램만 선택 -> Install / Update
